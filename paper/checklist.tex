\section*{NeurIPS Paper Checklist}

\begin{enumerate}

\item {\bf Claims}
\begin{itemize}
    \item[$\checkmark$] All main claims in the abstract and introduction are supported by experiments (Sections 4--6).
    \item[$\checkmark$] Limitations are explicitly discussed (Section 7: Discussion, ``Limitations'' paragraph).
\end{itemize}

\item {\bf Theory}
\begin{itemize}
    \item[$\checkmark$] Sen's meta-ranking formalization (Eq.~1--2) includes all assumptions.
    \item[$\checkmark$] Convergence of $\lambda_t$ proved via Banach Fixed Point Theorem (Appendix A, Theorem 1).
    \item[$\checkmark$] ESS verified via replicator dynamics with comprehensive numerical validation (Appendix A, Theorem 2).
\end{itemize}

\item {\bf Experiments}
\begin{itemize}
    \item[$\checkmark$] Eight environments tested: Cleanup, IPD, PGG, Harvest, Grid World, Climate Negotiation, Vaccine Allocation, Carbon Taxation.
    \item[$\checkmark$] Agent scales: 20, 50, 100, 500, 1{,}000 agents.
    \item[$\checkmark$] 7 SVO conditions $\times$ 10 seeds per condition (seeds: 0, 42, 123, 256, 999, 1337, 2024, 3141, 4269, 5555).
    \item[$\checkmark$] All error bars represent standard deviation across seeds; 95\% CI via bootstrap.
    \item[$\checkmark$] Statistical tests: OLS with HAC robust SE, Linear Mixed-Effects Models (LMM), Bootstrap CI.
    \item[$\checkmark$] Sensitivity analysis across all parameter conditions (Section 5.3, Appendix B).
\end{itemize}

\item {\bf Reproducibility}
\begin{itemize}
    \item[$\checkmark$] Code available at \url{https://anonymous.4open.science/r/EthicaAI}.
    \item[$\checkmark$] One-click reproduction: \texttt{python reproduce.py} (38 modules, 74 figures).
    \item[$\checkmark$] All hyperparameters specified in \texttt{simulation/jax/config.py} and Appendix B.
    \item[$\checkmark$] Random seeds explicitly set and documented.
    \item[$\checkmark$] Computing resources: single NVIDIA RTX 3090 GPU (12GB VRAM), $\sim$15 min for full pipeline.
\end{itemize}

\item {\bf Broader Impact}
\begin{itemize}
    \item[$\checkmark$] Positive: Framework for resource-aware moral reasoning in AI systems.
    \item[$\checkmark$] Risk: Meta-ranking mechanism could be repurposed for manipulative agent design.
    \item[$\checkmark$] Mitigation: Dynamic $\lambda_t$ is fully transparent and auditable; no opponent parameter access required.
    \item[$\checkmark$] Discussion of dual-use risks and moral relativism concerns included in Section 7 and Supplementary.
\end{itemize}

\item {\bf Safeguards}
\begin{itemize}
    \item[$\checkmark$] No human subjects involved; all experiments are computational simulations.
    \item[$\checkmark$] Human PGG reference data from published literature~(Fehr \& G\"achter, 2000).
    \item[$\checkmark$] No personally identifiable information in any dataset.
\end{itemize}

\item {\bf Licenses}
\begin{itemize}
    \item[$\checkmark$] Code released under MIT License.
    \item[$\checkmark$] No external datasets used; all data generated by simulation.
\end{itemize}

\end{enumerate}
